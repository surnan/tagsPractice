My HTML Cheat Sheet

Headings

Headings from `h1` to `h6` identify blocks and sections. Every page must have one `h1`

Paragraphs

Paragraphs of text go in `p` tags.

Unordered Lists

Unordered lists are marked with bullet poits and go in `ul` tags. Each list item goes in an `li` tag.

Types of lists

  • Unordered lists (`ul`)
  • Ordered lists (`ol`)

Ordered Lists

Ordered lists are numbered and go in `ol` tags. Each list goes in an `li` tag.

Anchors (redirection to URL that can be external)

Anchor tags create liks to other pages. Following link to Youtube. The URL goes in the `href` attribute. link

Setting `target="_blank"` will open the link in a new window. Link for more info. MDN for more details.

Anchor tags can also create links to a new location on the same page. Instead of putting URL int the "href" attribute, just use the "id" of the element you want to navigate to. This allows you to Jump to the top of page

Text Emphasis

Italic

The `em` element adds extra emphasis to the text within the opening and closing tags. This shows up as italic text in most browsers.

Bold

You can also use the `strong` element to emphasize text. This showsup as bold text in most browsers.
